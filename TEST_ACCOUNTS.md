## Заполнение базы

1. Применить схему БД (если ещё не применена):
   ```bash
   npx prisma db push
   ```

2. Запустить seed:
   ```bash
   npm run db:seed
   ```
   или
   ```bash
   node prisma/seed.js
   ```

После выполнения в БД будут: 3 ресторана с категориями и блюдами (с картинками), расписание, пользователи выше и несколько тестовых броней.
